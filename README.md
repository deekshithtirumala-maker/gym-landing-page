# Legend Fitness Center Website

Live demo
- https://legendfitnesscenter.netlify.app/

## Overview
A modern, premium dark-theme gym website built with pure HTML, CSS, and JavaScript (no frameworks required). The design is conversion-focused and responsive for mobile/tablet/desktop.

## Features
- Sticky navbar with smooth section scroll
- Full-screen hero with overlay and CTA buttons
- About section with icon highlights
- Programs grid with hover card effects
- Trainers section with profile cards
- Pricing plans with "Most Popular" badge
- Testimonials carousel (auto and manual)
- Photo gallery with zoom hover
- CTA section
- BMI calculator
- Contact form with "Send via WhatsApp" and "Send via Email"
- Google Map embed for location
- Sticky floating content and dark neon style
- Lazy image loading + scroll animations

## Getting Started
1. Clone or download this repository.
2. Open `index.html` in a browser (or run local server):
   - `python -m http.server 8000`
   - Visit `http://localhost:8000`

## Files
- `index.html` – page structure
- `styles.css` – design and responsive styles
- `script.js` – interactive behavior and form logic

## Customization
- Replace image URLs in `index.html` with proprietary fitness images.
- Update text/CTA, program titles, trainer info.
- Adjust `styles.css` color tokens (neon green/orange accent) to match brand.

## Map iframe fix
To use a working map embed, make sure the map source uses the Google Maps embed URL format (copy from map share -> "Embed a map").

## Notes
- No `npx` or React required.
- Works in modern browsers.
